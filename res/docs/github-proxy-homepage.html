欢迎使用Github-Proxy

基于Cloudflare Workers的Github文件代理加速工具


GitHub文件链接带不带协议头都可以，支持Release、Archive以及文件，右键复制出来的链接都是符合标准的。
Release、Archive使用CloudFlare加速，文件则使用JsDelivr加速
注意，不支持项目文件夹
